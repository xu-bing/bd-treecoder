课程内容

Course Content

Scratch图形化编程


Scratch是麻省理工学院的“终身幼儿园团队”开发的图形化编程工具。

不仅易于孩子们使用，又能寓教于乐，让孩子们获得创作中的乐趣。

Level1

三年级以上

丰富有趣的故事，浅显易懂的讲解，从零开始轻松入门Scratch3.0编程。学会创作动画、音乐、美术、游戏等作品。

Level2

三年级以上

跟随玩仔和周周进入代码世界去冒险，学习事件、循环、变量、逻辑判断等基本的计算机程序设计方面的知识。

Level3

三年级以上

踏上寻找十二神兽之旅，解锁克隆、列表、函数等高级的技能，学习游戏设计的思想，激发学生无穷的创意。

Level4

三年级以上

将Scratch和Picoboard相结合，利用各种传感器做出生动有趣的游戏。如水果电阻、植物大战僵尸、声控版碰碰车。

Arduino 硬件编程

Arduino是由一个欧洲开发团队于2005年冬季开发的一款便捷灵活、方便上手的开源电子原型平台。包含硬件和软件部分。

它简单易学，硬件操作十分简单，小孩子能轻松上手，结合Arduino与各种传感器， 设计好玩实用的作品。

Level1

四年级以上

学习常用的的Arduinot板、LED、舵机, 学习基础的电子电路知识和传感器知识，学习循环、条件、变量等基础编程知识。创作按键钢琴，投石机等有趣作品。

Level2

四年级以上

认识人体红外、手势传感器、温度传感器等进阶传感器，数码管和点阵屏等显示器，制作出电子时钟，迎宾机器人，扫地机器人等有趣的作品。

Level3

四年级以上

认识循迹传感器、倾角传感器、三维角度传感器、蓝牙模块和LCD等元器件，制作出电子时钟，迎宾机器人，扫地机器人等有趣的作品。

Level4

四年级以上

认识语音识别、指纹识别和WIFI等高级模块，了解生活中常见智能设备的硬件组成和设计原理，自己设计出能改善生活的智能设备。。

Python 代码编程

Python已经成为全球排名前四的语言，从云端、客户端，到物联网终端，python应用无处不在。相对于其他语言，Python更加易学、易读，非常适合快速开发。

通过学习，可参加全国IT类科技竞赛--蓝桥杯，以赛促学，为孩子的出国升学加分。

Level1

五年级以上

主要利用turtle海龟库，绘制各色的形状和图案，在充满趣味性的学习过程中逐步引导学员入门Python语言。

Level2

五年级以上

学习Python的基础知识，如字典，集合，函数，以及类和对象，继承与多态等，为之后进一步学习打好基础。

Level3

五年级以上

使用专业的PyCharm编写代码,通过学习Pygame框架，学会制作简单的2D小游戏，如“愤怒的小鸟”，“坦克大战”等。

Level4

五年级以上

学习网络传输的知识，学会从网上爬取所需的信息，制作网络爬虫，抓取学习资料。

NOIP算法编程

全国青少年信息学奥林匹克联赛（NOIP）由中国计算机学会统一组织，是一项面向全国青少年的信息学竞赛和普及活动。通过竞赛和相关的活动培养和选拔优秀的计算机人才。

通过竞赛获奖的同学，享有加分保送的优惠政策。

Level1

六年级以上

结合“多功能计算器”，讲解C++的变量、表达式、数据类型条件语句、循环语句等知识。

Level2

六年级以上

结合加密解密程序、文章着色程序等实例讲解递归、数组等知识。

Level3

六年级以上

结合排序程序、飞机大战游戏，讲解排序算法、结构体等知识。

Level4

六年级以上

结合题目介绍格式化输入输出、枚举、模拟、时间复杂度等知识。
